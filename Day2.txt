Forms :
----------

Template driven forms:
-> Angular infers the form onject from the DOM
-> App logic resides insde the template


Model driven forms:
-> Form is created programatically
-> App logic resides insde the component
-> Uses FormControl, FormGroup, FormBuilder

FormControl -> Returns the individual control
FormGroup -> Guops the individual control
FormBuilder -> Builds the group and returns the FormGroup
